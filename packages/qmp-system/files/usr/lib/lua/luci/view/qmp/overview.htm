About qMp

Quick Mesh Project (qMp) is a system for easily deploying Mesh/MANET networks using Wi-Fi technology.

qMp has been designed for being used in any scenario such as free community networks, corporative networks, large social events, quick network deployments, etc. You can read here about how it works.

The qMp firmware, based on OpenWrt GNU/Linux, works on many embedded WiFi network devices. You can get the compiled binaries for your device here. All the qMp source code is public, free and open, and can be accessed from the development website. If you want to join the project and contribute with your code, you can learn more on how to start contributing on qMp.


Visit our web page for more information: http://qmp.cat
